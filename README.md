# Modem-QAM-Simulation
 Simulation d’un modem QAM en Python : génération de symboles (avec code Gray), création d’une table de correspondance (bits, symboles, I/Q, énergie, phase), constellations IQ en émission/réception avec bruit (AWGN, phase), calcul BER/SNR, visualisation BER vs SNR, transmission d’image.
